# Fyrry_Fox_Fabric_1.20.6
***This document is written in Russian and English.***

## ENG
Fyrry_Fox_Fabric_1.20.6 is a mod for the corresponding version of minecraft.

This mod allows you to build your own FURRY VILLAGE.

At the moment, the mod has the following mechanics:
- The player can assign village centers (block.examplemod.center_furry_village)
- The player can spawn Furry Fox entities (item.examplemod.furry_fox_spawn_egg)
- The player can put beds in villages. Furry Fox will choose them as a place to sleep.
- The player can give Furry Fox the profession of Farmer with an iron hoe.
- The player can put composters in the village. Farmers can work around composters within a radius of 8 blocks. Farmers can plant, harvest, accelerate the growth of cultivated plants, and compost the excess. One Farmer is linked to one composter. One Farmer can work on different areas of composters.

## RUS
Fyrry_Fox_Fabric_1.20.6 - это мод для соответствующей версии minecraft.

Данный мод позволяет построить свою ФУРРИ ДЕРЕВНЮ.

На данный момент мод имеет следующие механики:
- Игрок может назначать центры деревни (block.examplemod.center_furry_village)
- Игрок может спавнить сущности Furry Fox (item.examplemod.furry_fox_spawn_egg)
- Игрок может ставить кровати в деревн. Furry Fox будут выбирать их в качестве места для сна.
- Игрок может выдать Furry Fox профессию Farmer с помощью железной мотыги.
- Игрок может ставить компостеры в деревне. Вокруг компостеров в радиусе 8 блоков могут работать Farmers. Farmers могут сажать, собирать, ускорять рост культурных растений, компостироват излишки. Один Farmer имеет привязку к одному компостеру. Один Farmer может работать на разных областях компостеров.
